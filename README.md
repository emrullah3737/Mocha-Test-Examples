Mocha JS Testing Examples
======================
Some testing examples for beginner

What's inside
----------------
Batteries included:
 - Mocha, Chai

Setup
-----
Clone the repository and install the dependencies.

    $ git clone https://github.com/emrullah3737/Mocha-Test-Examples.git my-project
    $ cd my-project
    $ npm install

Testing
------
If you want to test 

    $ npm test
